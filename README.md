# 🚀 Scalex - Insights & Blogs UI

A modern and responsive **Insights & Blogs Landing Page** built using **HTML5 & CSS3**.

This project is inspired by premium SaaS and modern agency website designs.  
The goal of this project was to practice:

- Modern UI Layouts
- Flexbox
- Card Design
- Hero Sections
- Navigation Bars
- Responsive Design Principles
- Clean Typography
- Professional Spacing System
